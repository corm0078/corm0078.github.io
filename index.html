Into the Maritimes

The Challenge

The challenge was creating icons that would represent the Eastern Canada but mostly the Maritimes these icons would become icons that will be used for touristic guides or websites.

The Solution

The solution was creating icons that could work as outlines or with fun colours. A modern twist was implemented with the flat design aspect. The line thickness was kept the same throughout to make the icons look very familiar. Colours were also chosen carefully to remind of the theme and keeping with familiar colors creating a strong bond between the icons. This familiar feel will allow the viewer to recognize the icons.

The Tea Expert

The Challenge

Creating an app that would appeal to the newer or older Davids Tea client. The app should reflect the company but also become its own brand as an extension of Davids Tea. The app should solve a problem with ease and pleasure while being user friendly.

The Solution

The Solution was creating an app with a simple tab bar that would allow the use to brew the perfect cup of tea. The App as three option brew,library and favorite to facilitate the brewing process.

Epax Slab

The Challenge

The challenge was creating a new slab serif that would make a great display font. This display font needed to capture the attention of the audience.

The Solution

The solution was creating lines that would remind of a traditional slab but with a new modern way the lines are very clean and the width will stay the same making the font family into a great new font.

MOGL

The Challenge

Creating an desktop application for MOGL. By pre-selling blocks of hours, MOGL gives business owners rare, affordable access to MBA-level business expertise - and eliminates MOGL time spent on chasing and collecting its own A/R. The problem, however, is that no system and time-tracker on the market effectively.

The Solution

The Solution was creating an app that would allow the user to track time against retainer at a glance. This was accomplish with flat hierarchy that would allow the user to see all the pertinant information at a glance.

UX/Web Design

I really enjoy web and user experience. I first realized the satisfaction of seeing someone use your website. I love trying to understand how the user think and how they will use the final product. When designing the user is always the main person on my mind. If the user is happy and has a understanding of the application then I have done my work. After working on something for so long there is nothing better then showing it to the target market/user to see what they think and how to improove something.
